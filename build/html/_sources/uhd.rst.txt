UHD Installation Guide
======================

This guide provides step-by-step instructions for installing UHD (USRP Hardware Driver) version 3.15.0.0 using a Bash script.

Prerequisites
-------------

Before running the installation script, ensure that you have the following:

- A Unix-based operating system (e.g., Linux, macOS)
- Git installed on your system
- CMake installed on your system
- Root privileges (sudo access) for the installation step

Installation Steps
------------------

Clone the UHD Repository
^^^^^^^^^^^^^^^^^^^^^^^^

The script starts by cloning the UHD repository from GitHub using the following command:

.. code-block:: bash

   git clone https://github.com/EttusResearch/uhd.git

This command creates a local copy of the UHD repository in a directory named "uhd".

Change Directory to the Cloned Repository
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

The script then changes the current directory to the cloned "uhd" repository using the ``cd`` command:

.. code-block:: bash

   cd uhd

This step ensures that the subsequent commands are executed within the UHD repository.

Checkout the Specified Version
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

To install a specific version of UHD, the script checks out the desired version using the ``git checkout`` command:

.. code-block:: bash

   git checkout v3.15.0.0

In this case, the script checks out version 3.15.0.0 of UHD.

Change Directory to Host and Prepare for Building
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

The script then navigates to the "host" directory within the UHD repository and creates a "build" directory:

.. code-block:: bash

   cd host
   mkdir build
   cd build

The "build" directory will be used to store the build files generated by CMake.

Configure, Build, and Install UHD
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Inside the "build" directory, the script uses CMake to generate the necessary Makefiles, builds the UHD library, and installs it:

.. code-block:: bash

   cmake ..
   make
   sudo make install

The ``cmake ..`` command generates the Makefiles based on the CMakeLists.txt file in the parent directory.

The ``make`` command builds the UHD library using the generated Makefiles.

The ``sudo make install`` command installs the built UHD library on your system. This step requires root privileges, so you may be prompted to enter your password.
